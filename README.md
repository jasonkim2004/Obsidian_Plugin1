# Image_Display

A simple Obsidian plugin that displays a floating image above your notes.

https://github.com/user-attachments/assets/183f9768-b72d-4124-8195-f65e124e16f3

## Features

- Show a floating image on top of the Obsidian workspace
- Use a vault-relative image path from the plugin settings
- Drag the image to move it around
- Resize the image with a corner handle
- Restore the image to its original size from the right-click menu
- Turn off the plugin from the right-click menu
- Fall back to a black box when no image can be loaded

## Usage

1. Place an image file inside your Obsidian vault.
```
  ex) Attachments/chiikawa.png
```

2. Open Obsidian settings.

3. Go to
```
  Community plugins > Obsidian Chiikawa
```

4. Enter the vault-relative image path.
```
  ex) Attachments/chiikawa.png
```

5. The image will appear as a floating element above your notes.

## Controls
- Drag the image to move it.
- Left-click the image to show the resize handle.
- Drag the resize handle to resize the image.
- Click outside the image to hide the resize handle.
- Right-click the image to open the context menu.
